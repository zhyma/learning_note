1. Euler-Lagrange modeling

  • The Lagrangian:
    L(q,q˙)=K(q,q˙)−P(q)+λTC(q,q˙)

    • K(q,q˙)=12q˙TM(q)q˙: the total kinetic energy, e.g. 12mx˙2+12my˙2, K≥0. 动能。
    • P(q): the potential energy, e.g. mgy, P might be negative (depends on the coordinate) (have a lower limit?). 势能。
    • C(q,q˙): constraints (not necessary， why?)

  • Eular-Lagrangian equations:
    ddt∂L∂q˙i−∂L∂qi=ui,  i=1,⋯,n.
    Lagrange: 在初始时刻建立坐标来标记质点ref
    Eular: 随着质点的变化以当前状态作为参考标记质点
    方程表明了两种坐标系如何转换
    ui is the generalized force performing work on the qi-th coordinate. (Gravity is not a gneralized force, counted by P)
    搞清这俩参照系怎么回事

  • 结论：M(q)q¨+C(q,q˙)+g(q)=u or M(q)q¨+C(q,q˙)+N(q)=τ

    • M(q)q¨: F=ma, positive definite
    • C(q,q˙)q˙: Coriolis force科里奥利力, F→=−2m(ω→×v→). M˙−2C is skew-symmetric (AT=−A)
    • g(q): gravity

2. State space

  • 简化问题以便求解：把高阶系统（多次求导，如速度加速度）表示为一阶系统（加速度是速度的差分，速度是位置的差分，将多个state合并写作向量形式，state之间的关系从等式组改写成矩阵形式）；若条件适当，将非线性系统转化为线性系统（x˙=f(x)->x˙=Ax+B）。期望的最终形态表示一个控制系统x˙=Ax+Bu。
    如y⃛+a1y¨+a2y˙+a3y=b0u为三阶系统。令state space formx=[yy˙y¨]T，则x˙=[y˙y¨y⃛]T
    y⃛=−a3y−a2y˙−a1y¨−b0u=[−a3−a2y−a1y][yy˙y¨]T+b0u
    x˙=[010001−a3−a2y−a1y]x+[00−b0]u
    即把一个三阶方程简化为了一阶方程，用矩阵表示了state space中state的内在联系（如导数组成的线性方程组）
  • Matlab ODE只可求解一阶ODE，必须将高阶系统转化为一阶系统才可求解

5. Stability

5.1. In general

A general nonlinear, time varying and autonomous system x˙=f(x,t) for time t and state vector x. (Images are from math24)

stable type condition
Stability, 稳定 Let x(t,x0) be a solution to the differential equation with initial condition x(0)=x0. A solution is stable if other solutions starting near x(t,x0) stay close to x(t,x0)
∀ϵ>0,∃δϵ>0,∥x0′−x0∥≤δϵ,⟹∥x(t,x0′)−x(t,x0))∥≤ϵ,∀t≥t0
在x(0)=x0附近时若收到扰动，结果还在其附近的受限范围内
Asymptotically stable, 渐进稳定 Stable and limt→∞(x(t,b)−x(t,a))=0 for b is sufficiently close to a, where a and b are two different initial states.
∃ϵ>0,∥x0′−x0∥≤δ,⟹∥x(t,x0′)−x(t,x0)∥→0 as t→∞
在x(0)=x0附近时若收到扰动，结果会逐渐回到x(0)=x0
Global asymptotically stability (G.A.S.) δ≈∞
在全局任意位置，都会去到x(0)=x0
Exponential stability (⟹ A.S.) ∃δ,c,λ>0:∥x0′−x0∥<δ→∥x(t,x0′)−x(t,x0)∥≤ce−λt∥x0′−x0∥
渐进稳定，接近速度用自然对数规定
  • linear system:
    • Asymptotically stable ⟹ G.A.S.
    • Exp. stable = A.S.
  • Nonlinear system:
    • A.S. ⇏ G.A.S.
  • General
    • Exp. stable ⟹ A.S.
  • 稳定之间的关系
    Stable →x(t) converges to xG Asymptotic stability →converges within time t Exponential stability

5.2. Stability of a linear system

  • A solution of the system x˙=ax, x(0)=x0 is x(t)=eatx0
    ↳A solution of the system x˙=Ax is x(t)=eAtx0
  • ODE判断稳定性的缺点：ODE计算耗时，并且需要计算无限的长度
  • 由矩阵特征值与特征向量可知，x˙=Ax可写成x˙=TΛT−1x
    x˙=TΛT−1x
    T−1x˙=ΛT−1x
    T−1dxdt=ΛT−1x
    dT−1xdt=ΛT−1x
    令z=T−1x，则有z˙=Λz，z˙i=λizi，一个可能解为zi(t)=zi(0)eλit。所以稳定性取决于λi的实部和0的关系。所以系统的稳定性取决于A 矩阵的特征值
  • 举例：二阶系统
    • 系统模型为mx¨+cx˙+kx=0 典型表达式x¨+2ζ0ω0x˙+ω02x=0（拉普拉斯变换后在s域上为s2+2ζ0ω0s+ω02=0），其中ω0是natural frequency，ζ0是damping ratio，参考。(拉普拉斯域中s2即为x¨，s为x˙）
    • 用x˙=Ax的形式可以表示做[x˙x¨]=[01−ω02−2ζ0ω0][xx˙]。则det(A−λI)=λ2+2ζ0ω0λ+ω02=0，
      解出eigen value是λ=−ζ0w0±|w0|ζ02−1。
    • 由上可知矩阵A的特征值是传递函数的极点。ζ=0共轭虚根无阻尼，0<ζ<1共轭复根欠阻尼，ζ=1两个相等负实根临界阻尼，ζ>1两个不相等负实根过阻尼）（知乎）。

5.3. Stability of a LTI system

The linear system x˙=Ax is

stable type requirement
asymptotically stable at the equilibrium iff Re(λi)<0, λi∈∀eigen value of A
marginally stable at the equilibrium iff Re(λi)≤0, λi∈∀eigen value of A
unstable iff ∃λi is eigen value of A, Re(λi)>0
  • LTI系统的Asmp. stable = Exp. stable（asyptotically, exponential）
