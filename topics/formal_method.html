2. Automata

2.1. A finite-state semi-automaton: A=(Q,Σ,δ)

  • Q: A finite set of states
  • Σ: A finite set of symbols (words，相当于自动机的“程序”)
  • δ:Q×Σ→2Q: A nondeterministic transition function
  • 没有指定开始状态，也没有指定终止状态

2.2. A finite-state automaton: A=(Q,Σ,δ,Q0,F)

  • Q0: An initial state
  • F: A set of final states
  • Büchi: An infinite word w=w0w1… is accepted if it results in a run to visit F infinitely often. （无限长的word，时不时就会经过终态）
  • Regular: A finite word w=w0w1…wn is accepted if it results in a run to ONE of the final states （有限长的word，最终抵达终态）
  • Deterministic finite automaton (DFA)
    Q0 is a singleton
  • L: Language
    The set of words accepted by the automaton.

  • Kleene star ∗ 闭包

2.2.2. ω-regular languages

  • Given a set Σ, the set Σω are obtained by concatenating words from Σ infinitely many times.
    e.g.: {0,1}ω={0,1,00,01,10,11,001,…}

3.1. Finite state transition system

  • A finite state transition system TS is a tuple
    TS=(S,Act,→,I,AP,L)
    • S: a finite set of states.
    • Act: a finite set of actions.
    • →⊆S×Act×S: a finite set of transitions.
    • I: a finite set of initial states.
    • AP: a finite set of atomic propositions.
    • L: S×2AP a labeling function.

3.2. Predecessors and successors

  • a-successor of s:
    Post(s,a)={s′∈S|s→as′}
    Post(s)=∪a∈ActPost(s,a)
  • a-predecessor of s:
    Pre(s,a)={s′∈S|s′→as}
    Pre(s)=∪a∈ActPre(s,a)
  • A state is terminal iff Post(s)=∅

3.3. Deterministic transition system (TS)

  • 两层含义

    • Action-determinisitic
      If |I|≤1 and for all s∈S,a∈Act, it holds that |Post(s,a)|≤1.
      For all actions and states, it has at most one suceessor state (至多有一个初始状态（为什么要限制I?），并且从任意一个state执行的任意一个action最多有一个后续状态)ch. 8
    • AP-determinisitic
      If for all A∈2AP the following conditions hold: |{s∈I|L(s)=A}|≤1 and for all s∈S,|Post(s)∩{s′∈S|L(s′)=A}|≤1
      每个AP（原子命题）至多有一个初始状态，并至多有一个后续(post)状态。需要确认?

  • Notation: for determinisitic transition function, T:S×Act→S

3.4. Interleaving of transition systems

  • TS1=(S1,Act1,T1,I1,AP1,L1), TS2=(S2,Act2,T2,I2,AP2,L2)
    T1||T2=(S1×S2,Act1∪Act2,→,I1×I2,AP1∪AP2,L)
    Transition relation → is defined by s1→a1s1′(s1,s2)→a(s1′,s2) and s2→a2s2′(s1,s2)→a(s1′,s2)
    L((s1,s2))=L(s1)∪L(s2)

3.5. Handshaking

  • TS1=(S1,Act1,T1,I1,AP1,L1), TS2=(S2,Act2,T2,I2,AP2,L2)
    T1||T2=(S1×S2,Act1∪Act2,→,I1×I2,AP1∪AP2,L)
    L(⟨s1,s2⟩)=L(s1)∪L(s2)
    Transition relation → is defined by
    1. If α∈Act1∩Act2, s1→α1s1′, and s2→α2s2′, then ⟨s1,s2⟩=L(s1)∪L(s2)→α⟨s1′,s2′⟩
    2. If α∈Act1\Act2, and s1→α1s1′, then ⟨s1,s2⟩=L(s1)∪L(s2)→α⟨s1′,s2⟩
    3. If α∈Act2\Act1, and s2→α2s2′, then ⟨s1,s2⟩=L(s1)∪L(s2)→α⟨s1,s2′⟩
