Formal method

  • A⟹B=¬A∨B
    • Example:
      A: If it is a bear
      B: then it can swim
      1. If A is false (it is not a bear), it might be able to swim (a fish) or not (a bird)
        It can swim -> B=T
        It cannot swim -> B=F
        A false statement implies anything
      2. If A is true (which it is a bear), it can swim (B=T)
    • A and B does not necessary to have causal relationship (example)
