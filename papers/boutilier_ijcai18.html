Planning and learning with stochasitc action sets

Craig Boutilier et al., IJCAI 2018

1. Model

  • n states S
  • Base actions Bs, for s∈S (all actions). For notational ease, assume that feasible action sets for each s∈S are identical, so Bs=B. Let |B|=m
    • The set of actions available at state s at any stage t is a random subset As(t)⊆B. Assume a family of action availability distributions Ps∈Δ(sB) These can depend on s∈S but are otherwise history-independent, hence Pr(As(t)|s(1),⋯,s(t))=Pr(As(t)|s(t)). Only actions k∈As(t) in the realized available action set can be executed at stage t.
  • The number of state-action pair is M=|S×B|=nm
    Feasible action sets for each s∈S are identical, Bs=B (allowing distinct base sets at different states has no impact on what follows)
  • Transition P:S×B→Δ(S). Use ps,s′k to denote the probability of transition to s′
  • reward function r:S×B→R. Use rsk to denote the accrued reward.
  • Discount is always fixed as γ,0≤γ≤1

3. Reformulation: convert to embeded MDP or compressed MDP

3.1. The embeded MDP

  • Embed the (realized) available action set into the state space itself. 将一个state根据可以执行的不同的action sets细分为不同state
  • State space: Se={s∘A:s∈S,A⊆B}, with s∘A having feasible action set A.
  • Transitions are defined as: ps∘A,s′∘A′k=P(s′∘A′|s∘A,k)=ps,s′kPs′(A′),∀k∈A
  • Reward: rk(s∘A)=rk(s) for k∈A
