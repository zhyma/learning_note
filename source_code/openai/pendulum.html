1. 输入，输出与模型

1.1. state, reward, done, {} = step(self, action)

  • Action:
    max_torqe = 2.

    Num Action
    0 [-max_torque, max_torque]

  • Observation (state):

    Num Observation Min Max
    0 cosθ -1 1
    1 sinθ -1 1
    2 θ˙ -8 8

  • Reward:
    r≤0, calculated by r=−costs=−(θ2+0.1θ˙2+0.001u2)
    u is the input torque u∈[−2,2]
    θ∈[−π,π], vertical up is 0.

  • Done:
    Always False

1.2. Model:

  • Variables

    Var value
    Gravity g 10.0
    Pendulum mass m 1
    Pendulum's length l 1
    Second between state updates Δt 0.05
    Input torque u Clampped to ensure the input is between [−2,2]
    Angular velocity θ˙ Clampped to [−8,8]

  • Kinematic model

    θ¨=(−3g2lsin(θ+π)+3ml2u)

  • Kinematics integrator:

    θ˙←θ˙+θ¨Δt
    θ←θ+θ˙Δt

  • Normalize θ
    θ←[(θ+π)mod(2π)]−π

2. 初始化

每次随机初始化θ∈[−π,π]，θ˙∈[−1,1]。
