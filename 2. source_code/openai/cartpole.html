1. 输入，输出与模型

1.1. state, reward, done, {} = step(self, action)

  • Action:

    Num Action
    0 Push cart to the left
    1 Push car to the right

  • Observation (state):

    Num Observation Min Max
    0 Cart Position -4.8 4.8
    1 Cart Velocity -Inf Inf
    2 Pole Angle -24 deg 24 deg
    3 Pole Velocity At Tip -Inf Inf

  • Reward:
    1 for every step taken, including the termination step.

  • Done:

    x或θ超出上下界时返回True。steps_beyond_done用于处理temination step的reward（None：pole未倒或处在termination step；>=0：中止）。

1.2. Model:

  • Variables

    Var value
    Gravity g 9.8
    Cart mass mc 1.0
    Pole mass mp 0.1
    Pole's length l 0.5
    Second between state updates τ 0.2
    Force F 10(right) or -10(left)

  • Kinematic model

    p=(F+lθ˙2sin⁡θ)/(mc+mp)
    θ¨=(gsin⁡θ−pcos⁡θ)/(4l/3−mpcos⁡θ2/(mc+mp))
    x¨=p−lθ¨cos⁡θ/(mc+mp)

  • Kinematics integrator:

    x=x+τx˙
    x˙=x˙+τx¨
    θ˙=θ+τθ¨
    θ=θ+τθ˙

2. 初始化

每次随机初始化所有状态在区间(−0.05,0.05)。
