    xt=Fxt−1′+But
    Σt=FΣt−1′FT+Q

    symbol variable
    xt state prediction
    F state transition matrix
    xt′ state estimate, refer to correction step
    B actuation function
    ut actuation
    Σt error covariance prediction
    Σt′ error covariance estimate
    Q process noise

  • Correction Step
    Kt=ΣtHT(HΣtHT+R)−1
    xt′=xt+Kt(zt−Hxt)
    Σt′=Σt−KtHΣt

    symbol variable
    zt observation
    H observation function
    R observation covariance

1. Example:

Assume a simple model of a robot with a range sensor mounted on a stationary platform. The sensor is initialized to 0, the error covariance is initialized to 3, the process noise is assumed to be 0.05, and the measurement noise is 0.1. Calculate the value of the Kalman Filter estimate of state (xt′ and Σt′) given the following data:

t xt Σt Kt zt xt′ Σt′
0 - - - - 0 3
1 1
1 2

In this example, we have:
x0′=0, Σ0′=3 are given.

  • For prediction step,
    For equation xt=Fxt−1′+But, We have: F=1(the form between estimation and prediction are the same, no need to transform), But=0 (no input).
    For equation Σt=FΣt−1′FT+Q, the process noise Q=0.05 is given. The error covariance initialization is provided (error covariance estimate at time 0), Σ0′=3. For steps after initialization, it is given by the previous correction step.
    So we simplified the equations to:
    {xt=xt−1′Σt=Σt−1′+0.05
  • For correction step,
    For equation Kt=ΣtHT(HΣtHT+R)−1, Σt is calculated from prection step. Observation function H=1 because of getting the range data directly from the sensor. R (measurement noise) is given as R=0.1.
    For equation xt′=xt+Kt(zt−xt), xt is from prediction. Kt is from the above equation. zt is the sensor's data (as provided).
    For equation Σt′=Σt−KtΣt, they are from previous equations.
    So we simplified the equations to:
    {Kt=Σt(Σt+0.1)−1xt′=xt+Kt(zt−xt)Σt′=Σt−KtΣt
  1. We can calculate x1 and Σ1 for step 1 by given x0′=0 and Σ0′=3. We don't need other variable for step 0.
  2. We can get K1 and x1′ at time t=1 by using x1and Σ1, and z1=1, which is given.
  3. From K1 and Σ1, we can get Σt′
  4. Now we finish step1, we have x1′ and Σ1′ to initiate at time t=2. Repeat step 1 with new value.
